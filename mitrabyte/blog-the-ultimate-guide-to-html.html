The Ultimate Guide To HTML

6TH 07 2023 FRONT-END DEVELOPER info-tech Professional.

HTML Basics

If you want to become a webmaster and learn how to create a website, you may find the prospect of getting to grips with HTML quite daunting.

This is something that cannot be avoided, however, as the vast majority of website landing pages that you visit will have been written and structured using HTML elements. In fact, HTML is now used by more than 74% of all known websites, while this language also helps to enhance everything from the design of your site to the quality of the content that it features.

In this guide, we will explore the basic principles of HTML and its potential applications, before looking at examples of the individual elements that you use when coding your website.

What is HTML?

In simple terms, HTML represents the standard markup language for creating web pages online. It stands for Hyper Text Markup Language, and its most primary function is to establish the structure, layout and presentation of individual landing pages. While web browsers do not directly display HTML language, it plays a pivotal role in helping to create a visible, accessible and easy to use site.

HTML is also underpinned by a number of individual elements, which gradually build web pages, structure the presentation of content and bring your website to life. These elements are created and contained within ‘tags’, which define alternate pieces of content such as headings, paragraphs, and similar examples.

We will explore these elements and their construction in further detail below, while also looking at how they can be personalized to introduce color, links, and variable typography to your website.

As we can see, the most common application for HTML is the design of the individual landing pages that make up your website. This is not the only application of the popular coding tool, however, understanding its additional uses will help you to get the most from the language as a new webmaster. So, here are some of the further ways in which HTML can be applied:

  1. Create customizable elements within an existing page. If you intend to allow blog posts comments or the publication of user-generated content on your website, you can use HTML code fragments to enable this. These elements will enable users to emphasize keywords, embed links, and format comments depending on the restrictions that you put in place as a moderator.
  2. Create additional content for email. Email also uses HTML as the language for rich text messages, whichfeature links, text, and a variety of other elements that cannot be featured in plain-text alone. So, if you are looking to share an ebook that relates to your website through email, you can utilize HTML to optimize the impact of your message.
  3. Understand offline help documents that are installed on your computer. Interestingly, HTML is used as the format for computer-based help documents that are accessible offline. Basic knowledge of HTML can, therefore, help you to understand issues with your hardware and resolve them quicker, which in turn may ensure that you are able to restore your website quicker in instances when it has gone offline.

HTML Page Structure

Before you can build out an HTML page, you need the basics in place. Typically, a page will be made up of three structural elements:

  • Header: The header contains content relevant to all pages on your site, such as a logo or website name, and a navigation system. The header is seen on each page.
  • Main body: This occupies the largest area on a web page. It contains content specific to the page being viewed.
  • Footer: Footer content usually includes contact information, a shipping address, or legal notices. Like the header, the footer appears on every page, but it’s positioned at the bottom.

Here’s what those basic structural elements look like when they come together:

                        
                            <header>
                                <!-- Header content goes here -->
                            </header>
                            
                            <main>
                                <!-- Main body content goes here -->
                            </main>
                            
                            <footer>
                                <!-- Footer content goes here -->
                            </footer>

You can put text or code here, like a Google Analytics tracking code for example. The main body of your page goes here. Fill it with text and images!

Here’s another example, using header tags and the paragraph tag to structure content aesthetically. Plus, we’ve thrown in a footer tag for content below the main body of a page:

                        
                            <header>
                                <h1>My First Heading</h1>
                                <p>Welcome to my website!</p>
                            </header>
                            
                            <main>
                                <p>This is the main body of your page.</p>
                            </main>
                            
                            <footer>
                                <p>Contact information could go here</p>
                            </footer>

HTML Tags

The starting point for any HTML code is individual tags, which can be used to create all crucial elements and help to structure your web pages.

Heading Tags

All online documents, including web pages, begin with a heading. These are constructed using HTML tags, with the language currently allowing for up to six variably sized elements that also enable you to structure your content with additional titles, subtitles, and highlighted lines of bold text. To start your heading, you simply prefix the relevant text with the <h1>, <h2>, <h3>, <h4>, <h5>, or <h6> tag depending on the desired size. You must then apply a closing tag at the end of the heading to encapsulate the text, and it will be displayed as follows in HTML format:

                        
                            <h1>This is heading 1</h1>
                            <h2>This is heading 2</h2>
                            <h3>This is heading 3</h3>
                            <h4>This is heading 4</h4>
                            <h5>This is heading 5</h5>
                            <h6>This is heading 6</h6>

Once confirmed, this will translate into the following aesthetic on your website’s landing page:

This is heading 1

This is heading 2

This is heading 3

This is heading 4

This is heading 5
This is heading 6

(h1 is shown bigger than h2 and so on.) (h1>h2>h3>h4>h5>h6)

Here, the various sizes of the variable headings are clear to see, as is the fact that the browser adds a line before and after the heading. You will also notice that the closing tag at the end of the heading text has a slightly different aesthetic, but we will cover this in the following chapter as we look to use tags to define specific elements.

Paragraph Tags

A similar principle is applied to starting paragraph tags, which are depicted by <p>. This allows you to structure your content and divide it into relevant paragraphs, which in turn translates into an easier and more seamless reading experience. Once again, the <p> tag should be placed at the beginning of the relevant text, before the alternate closing tag is applied at the end to complete the effect. Unlike heading tags, however, there are no numerical variations that can alter the size of the text featured in the paragraph.

For example:

                        
                            <p>Your First Paragraph</p>
                            <p>Your Second Paragraph</p>
                            <p>Your Third Paragraph</p>

Outside of the HTML format, this will break the text down as follows on your finished web page:

Your First Paragraph

Your Second Paragraph

Your Third Paragraph

Line Break Tags

The first examples represent the most basic HTML tags, but there are others that utilize a different format and alternative closing tags. Take line breaks, for example, which create a distinction at which lines of text are broken and continued on the following line. There is a core difference betweenline breaks and paragraphs in the field of HTML, as while the latter are standard block elements that contain text, the former creates separation within an existing <p> element.

Because of this, line breaks represent an empty element in HTML and are therefore not defined by either opening or closing tags. Instead, they are depicted by the <br> tag, which can be inserted into existing <p> elements to break up text and potentially highlight important pieces of information. The single space between the character and the forward slash is crucial, as otherwise, the tag is not recognizable in HTML format.

Here is an example:

                        
                            Good morning<br>
                            Many thanks for your inquiry, we will contact you if we require anything else.<br>
                            Best Regards<br>
                            Mr. Jones

Once applied, this will break the text up as follows:

Good morning,

Many thanks for your inquiry, we will contact you if we require anything else.

Best Regards

Mr. Jones

As you can see, you can apply line breaks as often as you like within an existing paragraph element, so long as they add value and make the information easier to digest. You will also see that the opening and closing paragraph tags remain unchanged, with line break tags used to amend the layout of the text included within.

This is one of the examples of how HTML tags can be used to alter existing elements, which plays a key role in defining the visual layout of your web pages and content.

Horizontal Lines

Similarly, there are other tags that can be used within the <p> element or to further separate bodies of text on your web page. One of the most widely used is the <hr> tag, which helps to create an empty element that draws a visual, horizontal line between alternate sections of an online document. You may want to position a line between two bodies of text, for example, in order to refocus the reader or simply introduce a new visual element. This is how you create such a break in HTML:

                        
                            <p>Your First Paragraph</p>
                            <hr>
                            <p>Your Second Paragraph</p>

Here, the breaking between the <hr> characters and the forward slash highlight the construction of an empty element, while once again there is no closing tag required to complete the effect. This will create the following visual:

Your First Paragraph

Your Second Paragraph

HTML Links

Links are an essential part of web pages, allowing users to navigate between different pages or external resources. In HTML, you can create links using the <a> (anchor) tag.

To create a link, you need to specify the destination URL using the href attribute and provide the link text within the opening and closing anchor tags. Here's an example:

    
        <a href="https://www.example.com">Visit Example.com</a>

In the above code, replace "https://www.example.com" with the actual URL you want to link to. The link text "Visit Example.com" will be displayed on the webpage as the clickable link.

HTML Lists

Lists allow you to organize and present information in a structured manner. HTML provides two types of lists: ordered lists and unordered lists.

  • Ordered List: An ordered list is created using the <ol> (ordered list) tag. Each list item is represented by the <li> (list item) tag. Here's an example:
    
        <ol>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ol>

  2. Item 2
  3. Item 3
  • Unordered List: An unordered list is created using the <ul> (unordered list) tag. Similar to the ordered list, each list item is represented by the <li> tag. Here's an example:
    
        <ul>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ul>

  • Item 2
  • Item 3

HTML Tables

Tables are used to organize data into rows and columns. In HTML, you can create tables using the <table> tag. Each row is represented by the <tr> (table row) tag, and each cell within the row is represented by the <td> (table data) tag. Here's an example:

    
        <table>
            <tr>
                <td>Cell 1</td>
                <td>Cell 2</td>
            </tr>
            <tr>
                <td>Cell 3</td>
                <td>Cell 4</td>
            </tr>
        </table>
Cell 1 Cell 2
Cell 3 Cell 4

HTML Forms

Forms allow users to input data and submit it to a server. In HTML, you can create forms using the <form> tag. Within the form, you can include various input fields, such as text fields, checkboxes, radio buttons, and more. Here's an example of a simple form:

    
        <form action="/submit" method="post">
            <label for="name">Name:</label>
            <input type="text" id="name" name="name">
            <br>
            <label for="email">Email:</label>
            <input type="email" id="email" name="email">
            <br>
            <input type="submit" value="Submit">
        </form>

In the above code, the form's action attribute specifies the URL where the form data will be submitted, and the method attribute defines the HTTP method to use (e.g., "post" or "get"). The form contains two input fields: a text field for the name and an email field for the email address. The submit button triggers the form submission.

HTML Comments

Comments in HTML allow you to add notes or explanations within your code that are not displayed in the browser. They can be helpful for yourself or other developers to understand and navigate through the code. Here's an example of an HTML comment:

    
        <!-- This is an HTML comment. -->

Comments start with <!-- and end with -->. Everything within these delimiters is considered a comment and will be ignored by the browser when rendering the webpage.

HTML Entities

HTML entities are special characters that cannot be directly written in HTML. Instead, they are represented by entity names or numerical codes. For example, to display the copyright symbol (©) or the less than symbol (<), you can use the respective HTML entities: &copy; and &lt;. Here's an example:

    
        <p>This is an example of an entity: &copy;</p>
        <p>This is another example using the less than symbol: &lt;</p>

HTML Semantic Elements

HTML5 introduced a set of semantic elements that provide meaning and structure to different parts of a web page. These elements help search engines, screen readers, and developers understand the purpose and organization of the content. Here are some commonly used semantic elements:

  • <header>: Represents the introductory content or a group of navigational links at the top of a page.
  • <nav>: Defines a section containing navigation links.
  • <main>: Represents the main content of the document.
  • <section>: Defines a standalone section within a document.
  • <article>: Represents a self-contained composition, such as a blog post or news article.
  • <aside>: Defines content that is tangentially related to the main content, such as sidebars or pull quotes.
  • <footer>: Represents the footer of a document or a section.

By using these semantic elements, you can enhance the structure and accessibility of your web page.

HTML Multimedia

HTML allows you to embed multimedia content, such as images, videos, and audio, into your web pages. Here are some examples:

Images

To display an image, use the <img> tag with the src attribute specifying the image source URL. You can also provide alternative text using the alt attribute for accessibility purposes. Here's an example:

    
        <img src="image.jpg" alt="Image description">

Replace "image.jpg" with the actual image source and provide an appropriate description in the alt attribute.

Videos

To embed a video, you can use the <video> tag. Specify the video source using the src attribute and include fallback content within the opening and closing tags. Here's an example:

    
        <video src="video.mp4" controls>
            Your browser does not support the video tag.
        </video>

Replace "video.mp4" with the actual video source. The controls attribute adds playback controls to the video player.

Audio

To embed audio, you can use the <audio> tag. Similar to videos, specify the audio source using the src attribute and provide fallback content within the tags. Here's an example:

    
        <audio src="audio.mp3" controls>
            Your browser does not support the audio tag.
        </audio>

Replace "audio.mp3" with the actual audio source. The controls attribute adds playback controls to the audio player.

HTML Character Encoding

Character encoding ensures that special characters and symbols are displayed correctly on your web page. The <meta> tag with the charset attribute specifies the character encoding for your HTML document. Here's an example:

    
        <meta charset="UTF-8">

The example above sets the character encoding to UTF-8, which supports a wide range of characters from different languages.
